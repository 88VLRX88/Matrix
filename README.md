# Matrix.py

Моя реализация Графов в мои 14 лет

## Что это

Библиотека для работы с графами, которую я написал в 14 лет. Чистый Python. Реализованны BFS, матрицы смежности, спецификации и честно сказать не помню что еще, но что-то еще.

## Возможности

- Создать матрицу любого размера
- Построить обход (arranging) от заданной вершины
- Получить спецификацию
- Настраивать значения на каждой вершине(через саму матрицу вроде или через пропертисы, не помню)
- Гениальнейший текстовый язык для запросов свойств (`getProperties_wr` и `setProperties_wr`)
- Думаю много чего еще можно найти, если изъебнуться, можно построить интересные проекты на этом.

## Как это выглядит

```python
from Matrix import Matrix

g = Matrix(5)
g.createMatrix()
g.marking([(0,1,1), (1,2,2), (2,3,3)])

levels = g.arranging(0)
g.showArranging(levels)
```

Вывод:
```
0 - [0]
1 - [1]
2 - [2]
3 - [3]
```

## Про язык запросов

Можно запросить свойства вершин через строку:

```python
g.vertexSetProperties([{"Name": "A"}, {"Name": "B"}])
g.getProperties_wr("0|1?Name")
```

Или установить:

```python
g.setProperties_wr("0|1!age=i'18|name=s'Vasya")
```
